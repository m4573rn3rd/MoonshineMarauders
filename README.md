# Moonshine Marauders WoW Addon

This is a basic World of Warcraft addon created by Gemini.

## Features

*   A graphical user interface (GUI) that can be shown or hidden.
*   A slash command `/mm` to toggle the GUI's visibility.
*   A persistent database (`MoonshineMaraudersDB`) to store addon data.

## Files

*   `MoonshineMarauders.toc`: The Table of Contents file that defines the addon for World of Warcraft.
*   `MoonshineMarauders.xml`: The XML file that defines the structure and layout of the addon's GUI.
*   `MoonshineMarauders.lua`: The Lua script that contains the addon's logic, including the slash command handler and event handling.

## Usage

1.  Place the `MoonshineMarauders` folder into your `World of Warcraft\_retail_\Interface\AddOns` directory.
2.  Launch World of Warcraft.
3.  In the game, type `/mm` into the chat to show or hide the addon's window.
